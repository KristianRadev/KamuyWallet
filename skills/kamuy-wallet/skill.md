# Kamuy Wallet

Gasless USDC wallet for AI agents.

## Overview

Kamuy Wallet is an OpenClaw skill that enables AI agents to make payments and manage wallet operations using natural language commands.

## Commands

- **Payment**: `pay`, `send`, `spend`, `buy`, `purchase`
- **Query**: `wallet`, `balance`, `policy`

## Installation

Requires `kamuy-cli` and `kamuy-steward` dependencies.

Run `kamuy init` on first install, then `kamuy unlock` to start.
